# Joe Campanale's Italian Wine Book

## Source
Images from Joe Campanale's comprehensive Italian wine reference book covering all major Italian wine regions.

## Coverage
- 20 Italian regions + Sicily/Pantelleria/Aeolian Islands
- Regions ordered: Valle d'Aosta, Liguria, Piemonte, Lombardy, Trentino-Alto Adige, Friuli Venezia Giulia, Veneto, Emilia-Romagna, Le Marche, Tuscany, Umbria, Abruzzo, Molise, Lazio, Campania, Basilicata, Apulia, Calabria, Sardinia, Sicily/Pantelleria/Aeolian Islands

## Files
- `producers_index_raw.md` — Transcribed producer index from book (names + page numbers)
- `producers_matched.csv` — Index matched to regions via research

## Process
1. Transcribe producer index from book images
2. Research each producer to determine region
3. Create producer pages in wiki/producers/
4. Update wiki/regions/ rollups
5. Regenerate wiki/index.md
