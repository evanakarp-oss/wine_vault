# raw/web_research/

Source-of-record notes for producer pages seeded from **web research** (not from
CSW / Raeders / DTE / FASS / Berserkers feeds). Each file captures the URLs and
key facts used to write a `wiki/producers/<slug>.md` page, so claims on those
pages are traceable per the CLAUDE.md anti-pattern ("don't claim coverage without
verifying in `raw/`").

These are hand-captured research notes, dated. They are NOT critic clippings —
`## Vinous Reviews` / `## Wine Advocate (Kelley)` sections still require real
Web Clipper saves under `raw/clippings/`.
