> I’m now authoritatively informed that it’s more like eight or nine million bottles. And do they control the entire process from harvest of grapes to release of champagne, or for instance buy fermented juice / still wine from farmers (eg negoce style)? I’d love to know how the assemblage is done to…

It’s not so complicated: they’ll figure out the blend, then rack the appropriate volume from tank a, tank b, tank c etc to a blending tank for a tirage. Then repeat. Each tirage is thus composed of the same blend. It’s the same for all large wineries. I’m sure Lafite don’t own a tank big enough to hold the entire production either, for example.
