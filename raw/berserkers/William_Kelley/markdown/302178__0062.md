The vintage chart used to be more detailed, but is it actually meaningful? Honestly, anyone who thinks you can generalize about vintages as a whole like that is so lost that I’m not sure there’s any way to help them. Roumier will still taste too young when Drouhin is fully mature; the rainfall that helped one vineyard in Beaune in a given vintage might not have fallen on the neighbor (I see this first hand); one producer didn’t beat mildew in 2018, another did; etc etc. In my view, a vintage chart can only give very broad brush strokes, and anything beyond that is peddling an illusion.
