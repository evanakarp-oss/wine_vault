Well, I think the region’s best producers would be right to be a bit more ambitious. If we could establish a virtuous cycle of better work leading to better valorization, it would be positive for the region. Equally, we don’t want to see some of the exaggerated market dynamics we’ve seen in other regions lately… so it’s a problem. But, the bottom line is that if we want to see the back of herbicides, and want to keep high density old vine goblet pruned vineyards from being restructured (an interesting question, actually), and for that matter a lower suicide rate among Beaujolais winemakers, then the wines need to be a bit more expensive.
