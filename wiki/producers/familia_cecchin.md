# Familia Cecchin

Winemaker: **Familia Cecchin**. Maipú (Mendoza). Category: **Certified Organic + Natural**.

## Approach

OIA certified. Pioneers in organic wines without sulfites since 1959. Native yeasts, no tartaric acid, no SO2. Historical reference point for the Mendoza organic movement.

## Grapes

Malbec, Cabernet Sauvignon

## Key Wines

- Malbec Reserva No Sulfites
- Cabernet Sauvignon Organic
