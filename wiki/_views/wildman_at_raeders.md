## Conflicts and resolutions

- **Méo-Camuzet conflict**: appears on both the KL paste and the Wildman paste. Wiki tag now lists both. Same pattern as the [[Grand_Cru_Selections#conflict-—-domaine-roulot|Roulot conflict]]. Real-world Méo-Camuzet has long been Wilson Daniels in many markets — possibly both KL and Wildman pastes contain stale or out-of-context entries. Pending resolution.
- **Pascal Jolivet resolved**: previously flagged as ambiguous on the KL view (KL listed "Domaine Jolivet" under Northern Rhône, while Raeder's "Pascal Jolivet Sancerre" is Loire). Wildman's portfolio explicitly lists "Maison Pascal Jolivet" under LOIRE VALLEY, matching the Raeder's bottles. The KL "Domaine Jolivet" Northern Rhône entry is likely a different producer or a paste error.
- **Larkmead** appears on both Polaner and Wildman pastes? Let me verify — Polaner had it. Actually no, Larkmead was a Wildman-only match here. (Polaner had Carlisle/Bedrock/Maybach, not Larkmead — re-checked.)

## False positives caught and excluded

- *Bodegas Fariña* (Spain, Wildman — Toro) ≠ "Farina | Lugana" at Raeder's — different producer (Italian Farina is Banville-imported)
- *Bacio* (Italy, Wildman — no region) → "Bacio Divino" (Napa proprietary red) — different producer
- *Bodegas Fontana* (Castilla La Mancha) → "Cascina Fontana" (Barolo) — surname collision
- *Greg Norman Estates* → "Dana Estates" / "Kinsella Estates" — generic "estates" token
- *Santi* (Wildman Valpolicella) ≠ Biondi-Santi (Montalcino) — different family
- *Tenuta dei Re* (Piedmont) → Castello dei Rampolla / Fattoria dei Barbi — generic "dei" token
- *Greg Norman / Estates* (AU/NZ/USA) → generic "estates" matching
- *Ridge Vineyards* also matched "Pine Ridge" (different Napa producer) — Pine Ridge entries excluded; only real Ridge (Monte Bello, Geyserville, Estate) counted

## Methodology

1. Source landed at `raw/wildman/portfolio_2026-05-21.md` (297 wine producers; spirits/whiskey/distilleries filtered out — 15 items skipped).
2. Page format: COUNTRY (or COUNTRY, REGION) line in UPPERCASE preceding each producer name (often suffixed " External link" which is stripped).
3. Cross-check script: same normalization + whole-token matching as previous importer views. Iterated GENERIC list to avoid over-filtering cult surnames; second pass surfaced Sylvain Cathiard which the first pass missed (had "cathiard" in the over-aggressive drop list).
4. Hand-audit each match against the Wildman region tag and Raeder's cuvée.
