# Raeders — Elegant Reds for Warmer Weather (2026-06)

Brief: **light-to-medium bodied, elegant reds for slightly warmer weather**
— the chillable / lower-tannin / higher-energy end of red. **No Pinot Noir.**

This is squarely Evan's existing lane: the cellar already holds
[[2023_marcel_lapierre_morgon|Lapierre Morgon]],
[[2018_jean_foillard_morgon_eponym|Foillard Morgon]],
[[2021_daniel_bouland_morgon_pre_jourdan|Bouland Morgon]],
[[2022_domaine_chapel_fleurie_charbonnieres|Chapel Fleurie]],
[[2020_arianna_occhipinti_il_frappato|Occhipinti Frappato]] and an
[[2022_gustinella_jungimmune_etna_rosso|Etna Rosso]] — so this list both
**restocks** proven favorites and **extends** into adjacent terroir-reds
(Cab Franc, Mencía, Blaufränkisch, Barbera) that fit the same
sense-of-place + tension axis.

## Skipped (and why)

- **Beaujolais Nouveau** (Duboeuf, Bouchard-Aîné, Durdilly, La Madone) +
  **Louis Jadot Beaujolais-Villages** — generic / off-curation; not the
  grower-cru style Evan buys.
- **Braida "Bricco dell'Uccellone" / Coppo "Pomorosso" Barbera** — excellent
  but barrique-rich and full-bodied; off the *light/elegant* brief.
- **Dal Forno / Quintarelli Valpolicella** — structured, concentrated and
  pricey (Evan already owns a [[2009_romano_dal_forno_amarone_della_valpolicella_vigneto_di_monte_lodoletta\|Dal Forno Amarone]]);
  not a warm-weather chiller.
