# Burgundy — Best Value

Burgundy isn't dead at retail, it's just been pushed to the edges. The shape of the value frontier in 2026:

- **Côte de Nuits north**: Marsannay (Chenôve / Marsannay-la-Côte / Couchey), Fixin. Pre-1er-Cru promotion window still open through 2024 vintage.
- **Côte de Beaune south**: Auxey-Duresses, Saint-Romain, Santenay, Maranges.
- **Côte Chalonnaise**: Mercurey, Rully, Givry, Bouzeron (Aligoté).
- **Mâconnais**: Mâcon-Villages, Pouilly-Vinzelles/Loché/Solutré.
- **Côte d'Auxerre / Yonne fringes**: Saint-Bris (Sauvignon), Irancy (Pinot), Chablis 1er Cru (still a value lever vs. Côte d'Or whites).

Frame:
1. Anchor each appellation in **one Kelley pick** and **one CSW champion**.
2. Cross-check the vault for what's currently in DTE / Raeders / CSW.
3. Skip generic mid-tier village wines from prestige communes — that is where Burgundy is dead.

## Anti-patterns for Burgundy value hunting

- Don't buy generic village Gevrey, Vosne, Chambolle, Pommard, Volnay below 1er Cru level — that is where the value is gone.
- Don't pay 1er Cru prices for fragmented sub-parcels of low-rep lieux-dits.
- Don't buy "Bourgogne Rouge" from prestige négoces — Bourgogne from a real grower (Pataille's Chapitre, Lafouge's Bourgogne, etc.) on a real terroir is the bet.
- Don't ignore Marsannay 2022/2023 right now — the pricing reset is months away.
