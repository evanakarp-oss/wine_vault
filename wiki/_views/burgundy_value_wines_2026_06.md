# Burgundy Value Wines — WB Thread Cross-Reference (2026-06)

Cross-reference of a WineBerserkers "favorite value producers" thread (the ask:
Côte de Beaune, ideally Volnay / Montrachet, $50–$150 reds and whites) against
the vault. Three buckets: **already covered**, **gap candidates that fit Evan's
filter**, and **skip / generic-tier**. Evan's Burgundy filter is terroir-driven,
biodynamic-leaning, grower-scale — so the thread's negociant suggestions get
deprioritized even when the wine is fine.

The thread is unusually well-aligned with the vault's taste: most names are
grower-domaine, sub-$150, site-transparent. The gap list below is a strong
buy/onboard shortlist.

## 🎯 Gap candidates — fit the filter, worth onboarding

Ranked roughly by fit + how hard the thread pushed them. All grower-scale,
terroir-driven, sub-$150. Bullseye for the Volnay/Montrachet/Beaune ask are
flagged.

### Tier 1 — strong fit, multiple endorsements

- **Michel Lafarge** — Volnay, biodynamic, iconic restrained. *Bullseye (Volnay).*
  wbkeene's "LaFarge Vendanges [sélectionnées]." Textbook Evan style.
- **Bruno Clair** — Marsannay / Savigny La Dominode, domaine-scale classic.
  Named 3× (Dominode, the value benchmark of the thread).
- **Dureuil-Janthial** — Rully reds **and** whites. Endorsed 3× ("hits way above
  its price," 1er Margotes ~$100, Chêne lieu-dit ~$70). High-value white + red.
- **Hubert Lamy** — St-Aubin, high-density terroir whites. *White, Montrachet-adjacent.*
  wbkeene + Robert.A.Jr.
- **William Fèvre** — Chablis (Montée de Tonnerre, Les Clos). *White.* Steve_McL's
  "get you some 2020." Top-tier Chablis, fills a vault Chablis gap.
- **Marc Colin** — St-Aubin / Montrachet-adjacent grower whites. *Bullseye (Montrachet).*
  Gr3_gor_N. Father of PYCM (whom the vault already has).
- **Ghislaine Barthod** — Chambolle; Bons Bâtons the value benchmark (wbkeene).
  Top grower, on-filter.

### Tier 3 — natural/Beaujolais adjacency (on-filter but off-ask)

- **Jean Foillard** — Morgon Côte du Py, Gang-of-Four natural. Strongly on-filter
  (biodynamic/natural); not Côte de Beaune, but vault keeps a Beaujolais section.
  Barry_Lipton's "best Beaujolais I've ever had."

## 🚫 Skip / borderline (against the filter)

- **Louis Jadot** (Beaune), **Faiveley** (Mercurey) — large negociant-tier houses.
  Wines are competent and Faiveley's Mercureys are genuine value, but generic-leaning;
  the vault deliberately omits Jadot (cf. it carries [[louis_latour]] only as a Raeders
  inventory page). Onboard only if a specific bottling earns it.
- **Coquard-Loison-Fleurot** — "glossy, some oak… forward fruity" per the thread.
  Fun but modern/less site-transparent — lower priority against the sense-of-place axis.
- **Guillemot (Savigny 1ers)** — Stan_Y's pick; distinct from the vault's
  [[domaine_guillemot-michel]] (Mâcon-Viré). Obscure; confirm identity before onboarding.
