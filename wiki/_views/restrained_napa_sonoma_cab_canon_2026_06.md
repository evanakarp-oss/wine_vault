# Restrained / Elegant Napa & Sonoma Cabernet — House-Style Canon

The axis (per CLAUDE.md, flipped 2026-05-29): **sense of place + tension** —
dry-farmed or low-intervention, mountain or cool-site, lower-alcohol, site-transparent,
built to age. The opposite of the opulent cult tier. Evan's stated north stars are
**Dunn, Corison, La Jota, Dalla Valle, Ridge Monte Bello**; this canon widens that
to the full Napa+Sonoma restraint set, grouped so you can shop by terroir.

✅ = has a vault producer page · ◐ = referenced across the vault, no page yet

## Core shortlist (if you buy from one bracket, start here)

| Producer | Where | The one-liner |
|---|---|---|
| **Dunn Vineyards** | Howell Mountain | The austerity benchmark — low-alcohol, near-immortal mountain Cab. |
| **Corison** | St. Helena (valley) | Cathy Corison; dry-farmed Kronos, early-picked, <14%, the elegance icon. |
| **Mayacamas** | Mt. Veeder | 1,800–2,400', CCOF organic, "marathon-runner" tannins, 10–20 yr. |
| **Ridge Monte Bello** | Santa Cruz Mtns | The cool-climate Cab benchmark; structured, ageless. |
| [[mount_eden_vineyards\|Mount Eden]] ✅ | Santa Cruz Mtns | Dry-farmed Franciscan shale, ~2,000', La Questa lineage. |
| **Philip Togni** | Spring Mountain | Austere, tannic, classic — a back-vintage benchmark. |
| **Spottswoode** | St. Helena | Organic/biodynamic, restrained, ageworthy, <14%. |
| ◐ **Bedrock (Montecillo)** | Moon Mtn, Sonoma | Old-school mountain Cab off dry-farmed late-'60s vines. |

## Napa — mountain (the heart of the style)

**Howell Mountain** — iron-fisted, structured, slow:
- **Dunn Vineyards** — the benchmark.
- **La Jota** — Evan reference-set; classic Howell structure.
- **Viader** — terraced steep slopes, Bordeaux-leaning (DARE = value 2nd label).
- **Robert Craig** — mountain-Cab specialist, value (also Mt. Veeder / Atlas Peak).

**Mount Veeder** — high, cool, volcanic:
- **Mayacamas** — the godfather.
- **Random Ridge** — dry-farmed, natural-leaning (unverified — confirm before trusting).
- **Brandlin** — old-vine heritage Veeder.
- **Progeny** — restrained Veeder estate.

**Spring Mountain** — cool, forested, slow-ripening:
- **Philip Togni** · **Smith-Madrone** (dry-farmed) · **Stony Hill** (historic, now Cab too) ·
  **Cain** (Cain Five) · **Two Terroirs** (off-radar, back-vintage).

**Diamond Mountain** — volcanic, firm:
- **Diamond Creek** (classic) · **Dyer** (2.3 ac, <300 cs micro-cuvée) · **Reverie**.

**Pritchard Hill** (the restrained end only):
- **Chappellet** "Signature" / Pritchard Hill — hillside, classic, ageworthy.
  (Skip the Pritchard Hill cult tier: Colgin, Bryant, Ovid, Continuum-adjacent opulence.)

## Napa — valley floor but restrained / dry-farmed / classic

- **Corison** (St. Helena) — dry-farmed Kronos & Sunbasket; the standard-bearer.
- **Spottswoode** (St. Helena) — organic estate, restrained.
- **Frog's Leap** (Rutherford) — dry-farmed, organic; John Williams; gentle, savory.
- **Heitz Cellar** (St. Helena) — Martha's Vineyard; traditional, long-aging, minty-classic.
- **Forman** (Ric Forman) — restrained, Bordeaux-trained classic.
- **Grgich Hills** (Rutherford) — dry-farmed organic/biodynamic; high-acid, ageworthy.
- **BV Georges de Latour** (Rutherford) — Tchelistcheff legacy; old-school benchmark, drink-now back-vintages.
- **Inglenook** (Rutherford) — Bordeaux-styled classic under Coppola/Moueix-era hands.
- **Dominus** (Yountville / Napanook) — Christian Moueix; restrained Bordeaux hand (pricier).
- **Matthiasson** (Napa) — Steve Matthiasson; the balance-movement standard-bearer.
- **Ashes & Diamonds** (Napa) — retro-restrained Bordeaux style (Matthiasson-consulted).
- **Maître de Chai** "Gala Mountain" (Mt. George) — biodynamic, ~1,000', restrained new-Cal.

## Sonoma — Mayacamas / Moon Mountain spine + high elevation

**Moon Mountain District** (Sonoma-side Mayacamas — the Bedrock corridor):
- ◐ **Bedrock** — Montecillo Vineyard (dry-farmed, late-'60s vines) + the Heritage field blend.
- [[far_mountain\|Far Mountain]] ✅ — Rodrigo Soto; `Fission` from ~50-yr never-irrigated vines.
- **Hamel Family Wines** — **Demeter biodynamic**, dry-farmed since 2018; Nuns Canyon 1,200–1,600' volcanic; explicit "less is more" restraint. One of the strongest farming-driven fits in Sonoma.
- **Monte Rosso bottlings** — 1,300', 1940 Cab vines, high-acid volcanic; chase **Bedrock**, **Carlisle**, **Louis Martini** vineyard-designates.
- **Stone Edge Farm** (Silver Cloud, Coturri-farmed organic) · **Lasseter** · **Repris** · **Petroni**.

**Sonoma Mountain:**
- **Laurel Glen** — ~1,000', organic/Coturri-farmed, own "Laurel Glen clone"; elegant + firm, off-radar; library releases turn up.

**Rockpile / far-north high elevation:**
- **Rowen** — Cooley Ranch up to 2,040'; rocky, structured Bordeaux blend.

## Wine Berserkers community signal (search-recovered 2026-06)

Could not run the WB scrape pipeline (forum 403s the egress here), so this is
**search-recovered, not a full ingest** — no producer-frontmatter signals were
written. The hivemind validates this canon almost exactly:

- **Poll — [Berserkers Favorites: Cal Cab](https://www.wineberserkers.com/t/berserkers-favorites-cal-cab-producers-poll-results/111726)** (74 ballots, 7-4-3-2-1): Ridge **178** (runaway #1); Shafer 62 / Schrader 62 / Myriad 57 (*cult tier = skip*); then **Spottswoode / Dunn / Dominus 38** each; Corison 16. The restrained names that broke through: **Ridge, Spottswoode, Dunn, Dominus.**
- **[Sane, old-school Napa Cab producers](https://www.wineberserkers.com/t/sane-old-school-napa-valley-cab-producers/154981)** (quality + sane price + 20-yr track record): **Dunn** (top), **Corison**, **Philip Togni** (Tanbark Hill = entry), **Smith-Madrone**, **Mayacamas** ("makes Corison look like Helen Turley" — the most austere), **Chappellet**, **Frog's Leap**, Spring Mountain Vyd + historic houses (Mondavi, Krug, Martini, Stag's Leap).
- **[Lean California cabernets](https://www.wineberserkers.com/t/lean-california-cabernets/320408)** (2024): **Matthiasson** Napa 2020, **Arnot-Roberts** Clajeux 2019 — the modern lean benchmarks.
