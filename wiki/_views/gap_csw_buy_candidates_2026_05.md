# CSW Buy Candidates — Cellar Gap Analysis (2026-05)

Ranked picks from CSW's 143 dedicated-champion producers ([[gap_csw_championed|full list]])
that Evan doesn't own, cross-checked against:

1. **Cellar composition** — what's underweight vs. what's already deep.
2. **Evan's curation filters** (CLAUDE.md) — Loire / Mosel / Burgundy / Piedmont
   terroir-driven biodynamic-leaning growers; WK-style undercovered Bordeaux;
   grower / late-disgorged Champagne; no generic mid-tier.
3. **CSW exclusivity** — flagged if also at DTE / Raeders / FASS, since those
   give alternative paths.

**grower / biodynamic Mosel**.

## Order of operations

If buying one mixed case from CSW: 3× Baudry Chinon, 2× Gonon Saint-Joseph,
2× Falkenstein, 2× Schaefer or Busch, 2× Brovia or Esmonin, 1× Peybonhomme.
